get_explore_status(**kwgs)[source]¶

获取探索运行信息

获取探索运行状态和相关信息

Parameters

  • engine_key (str) – 引擎 key

  • explore_id (int) – 探索 ID，参见 AutoMLService.start_exploration 返回值

Returns

ret['currentStage'] 表示当前 stage ，ret['currentProgress'] 当前 stage 进度，ret['status'] 状态信息。

其中 currentStage 取值：

["uncreated", "create", "dataImport", "instanceExplore",
"instanceProcess", "modelBuild", "publish"]

其中 status 取值：

['instanceProcess_succ', 'instanceProcess_terminate', 'instanceProcess_running',
'instanceProcess_fail', 'modelBuild_ready', 'modelBuild_running', 'modelBuild_terminate',
'modelBuild_succ', 'modelBuild_fail', 'publish_succ', 'publish_fail']

Return type

dict
